Grad Slate Sync - UpdateSlate Endpoint

UpdateSlate Endpoint

Use the following endpoint to update Slate records:

/UpdateSlate

Parameters:

  • SlateGuid: VARCHAR(20)
  • BUID: VARCHAR(20)

Example Request:


POST /UpdateSlate
{
    "SlateGuid": "your_slate_guid",
    "BUID": "your_bu_id"
}
